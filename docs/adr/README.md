# ADRs

Use este diretório para registrar decisões arquiteturais.

## Nome do arquivo

`ADR-XXXX-titulo-curto.md`

Exemplo: `ADR-0001-modular-monolith-first.md`

## Quando criar ADR

- Mudança de estratégia arquitetural
- Nova dependência central
- Alteração de boundary entre módulos
- Decisão com impacto de longo prazo em custo, risco ou velocidade

## Template

Use `docs/adr/ADR-TEMPLATE.md`.
